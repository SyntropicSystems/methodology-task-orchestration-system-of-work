# Integration

## Purpose

The integration stage contains completed tasks undergoing verification, merge, release, and final integration into the system. This includes system-of-work review and knowledge capture.

## Entry Criteria

- Implementation completed and verified
- All expected outputs delivered
- Task moved from current with completion event in ledger
- Ready for independent verification

## Exit Criteria

- Task independently verified against acceptance criteria
- Merged, released, or integrated as appropriate
- System-of-work review completed
- Knowledge captured and documented
- All follow-up tasks created
- Task ready to move to archived
