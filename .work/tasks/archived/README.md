# Archived

## Purpose

The archived stage contains fully completed and closed tasks. Tasks here have been verified, integrated, and all follow-up work created.

## Entry Criteria

- Task independently verified against acceptance criteria
- Integration completed (merged, released, or deployed)
- System-of-work review completed
- All follow-up tasks created
- Knowledge captured and documented
- Final verification completed

## Exit Criteria

- Tasks remain in archived as permanent record
- Archived tasks are read-only
- Ledger provides complete provenance trail
