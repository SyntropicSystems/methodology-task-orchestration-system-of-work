# Current

## Purpose

The current stage contains tasks that are actively being worked on. Tasks here are in active implementation with a committed DRI.

## Entry Criteria

- Task picked up from next queue
- DRI actively working on implementation
- Implementation start event logged in ledger
- All dependencies satisfied or in progress

## Exit Criteria

- Implementation completed
- All expected outputs delivered
- Documentation created or updated as required
- Task ready for verification and integration
- Implementation completion event logged in ledger
